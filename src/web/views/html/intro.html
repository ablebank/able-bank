P2P Matching Engine
투자/대출 매칭 엔진이 대부분의 투자/대출을 담당하지만, 발생하는 다양한 금융 업무 수요는 금융 스마트컨트렉트를 통해 진행됩니다. 장기 대출, 고액 대출, P2P 대출, 담보 대출, 신용 기반 대출이 해당 됩니다.
ABLE 외부 정보를 기반으로 하는 담보 대출과 신용 대출은 ABLE 메인네트워크 개발 이후 다양한 연동을 통해 진행될 예정입니다.
초기 금융 스마트컨트렉트는 ABLE 에서 제작을 하고, 점차 금융 스마트컨트렉트 프레임워크를 기반으로 다양한 금융 상품 개발자가 금융 스마트컨트렉트를 제작할 수 있는 환경을 제공










Decentralized Exchange
ABLE Ecosystem 은 크게 ABLE 계좌(Account)를 중심으로 금융, 자산 관리, 마이크로 페이먼트(Micro
Payment)를 포함한 3rd 파티 지원으로 구성 됩니다. 다양한 암호화폐 기반의 스마트컨트렉트를 저장하기 위해
서 DEX(Decentralized Exchange : 탈중앙화 분산거래소) 기능을 추가합니다. DEX 를 기반으로 자산 관리 및
금융 스마트컨트렉트 제작이 가능한 프레임워크를 배포하여, 다양한 상품개발자가 금융 상품을 개발할
수 있는 환경을 제공합니다. 이를 기반으로 이용자들은 다양한 금융 상품을 사용할 수 있고, ABLE project
는 암호화폐 금융 플랫폼으로 진화하고, 마이크로 페이먼트를 포함한 외부 소통을 위한 API 개발을 통해 에코시















Easy Transfer
암호화폐의 주소 길이는 해시충돌과 해킹을 대비하기 위해, 비트코인-34글자 , 이더리움-65글자 의 길이를 갖고 있습니다. 긴 주소를 중심으로 송금 서비스를 이용하는 것은 사용자에게 번거로운 일입니다. 암호화폐 송금이 대중들에게 이용되기 위해서는 간편화된 주소를 도입하는 것이 필요합니다. 웹어플리케이션을 이용할 때IP주소대신기억하기쉬운도메인주소를이용하는것과유사한원리입니다.
